## 设计初衷

- 提供一个高效的 `React` 开发框架，代替 umijs、icejs 等，主要是面向简洁化，不需要一些花里胡哨的配置

- 实现一个更加适合自己开发习惯的一款定制化 `React` 框架

## 快速开始

```shell
yarn install lyr-cli -D
```
